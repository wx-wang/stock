# 产品需求文档 (PRD)

> 版本: v1.0 | 日期: 2026-07-03 | 状态: 设计中

---

## 一、功能概述

### 1.1 定位

**趋势分析** 是股票分析系统的第四层防线。在行业轮动 → 拥挤度过滤 → 个股筛选之后，对候选股票进行趋势确认——确保只参与处于右侧趋势的标的，避免左侧抄底和追高过热。

### 1.2 设计依据

参考趋势动物（Nick）趋势交易体系 + 欧奈尔（William O'Neil）CAN SLIM 方法的趋势部分。核心理念：**价格是第一性原理**，成交量、基本面等为辅助归因。

### 1.3 四大核心指标

| 指标 | 含义 | 类比 |
|---|---|---|
| **趋势温度** | 价格趋势的冷热状态（冻↔沸，9档） | 告诉你"能不能做" |
| **趋势节气** | 右侧趋势的发展阶段（立夏→大暑→立秋） | 告诉你"做到哪了" |
| **趋势强度** | 多周期收益率在全市场中的百分位排名（0-100） | 告诉你"选哪个" |
| **均线排列** | 价格与 MA5/MA10/MA20/MA60 的位置关系（0-4分） | 底层核心数据 |

### 1.4 完整决策链

```
CAPM 行业轮动   →  选行业
拥挤度          →  避开过热行业
行业掘金个股推荐 →  行业内挑好股票
趋势分析        →  确认趋势是否还在右侧  ← 最后一道闸门
```

## 三、指标说明

### 3.1 趋势温度（9 档）

| 温度 | 含义 | 交易指令 |
|---|---|---|
| **沸** | 均线完美多头 + ATR 显著放大 | 分段止盈 |
| **热** | P > MA5 > MA10 > MA20 > MA60 | 持有/开仓 |
| **温偏热** | 均线多头但价格跌破 MA5 | 注意，暂不开新仓 |
| **温** | 价格跌破 MA10 但 MA20 完好 | 关注，准备开仓 |
| **温偏凉** | 价格在 MA20-MA60 间摩擦 | 警惕，不出不入 |
| **平** | 均线纠缠，无方向 | 空仓/等待 |
| **凉** | 价格跌破 MA20 | 已清仓 |
| **寒** | 完美空头排列 | 空仓 |
| **冻** | 完美空头 + 下跌加速 | 空仓，不抄底 |

### 3.2 趋势强度（0-100）

```
综合强度 = 10日ROC分位×0.2 + 60日ROC分位×0.5 + 250日ROC分位×0.3
```

| 分数 | 含义 |
|---|---|
| 90-100 | 领头羊级 |
| 70-90 | 强势 |
| 50-70 | 中上 |
| 30-50 | 中下 |
| 10-30 | 弱势 |
| 0-10 | 极弱 |

### 3.3 趋势节气（状态机）

```
无右侧 → 立夏（首次进入完美多头）
       → 夏至（站稳5天 + MA20-MA60>0.5%）
       → 小暑（10天 + 短线间距加速）
       → 大暑（15天 + ATR>1.5倍）
       → 立秋（跌破MA10 或 温度转平）→ 重置
```

节气只进不退，立秋后归零。

### 3.4 均线排列评分（0-4）

4 对比较，每成立得 1 分：

| 比较 | 含义 |
|---|---|
| P > MA5 | 价格在超短线均线之上 |
| MA5 > MA10 | 超短线动量向上 |
| MA10 > MA20 | 短线趋势传到中线 |
| MA20 > MA60 | 中线确认长期（最关键） |

## 四、数据范围

- **覆盖面**：全市场 A 股（~5000 只），不含北交所
- **数据源**：`stock_basic`（列表 + 名称）+ `daily`（日线）
- **更新频率**：每天首次访问自动重建，后续走缓存
- **历史窗口**：列表 = 120 天，详情 = 250 天

---

## 五、用户故事

1. **快速扫描**：打开页面 → 看到全市场按温度排序的列表 → 一眼看到哪些在热/哪些在寒
2. **筛选**：搜索"茅台"→ 看茅台温度=热 → 点进去看详情
3. **趋势判断**：详情页看到 MA20 即将金叉 MA60 → 节气刚从立夏转夏至 → 确认可以参与
4. **风险检查**：hover 折线图看到前天的温度是温偏热 → 趋势结构没被破坏过
5. **卖点确认**：详情页显示"⚠️ 距立秋触发线仅差 0.5%"→ 准备减仓
