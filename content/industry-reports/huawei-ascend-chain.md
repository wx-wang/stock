# 华为昇腾链产业地图深度分析报告

> 生成日期：2026-07-13  
> 分析框架：A股产业地图六问 + 卖方叙事雷达 + 关系证据等级  
> 研究对象：华为昇腾链、Atlas/超节点、国产 AI 算力、A股可投资性映射  
> 结论用途：研究支持，不构成证券投资建议

---

## 0. 核心结论

华为昇腾链应该拆成两张图看。

第一张是“真实产业链”：昇腾 NPU、Atlas 硬件、CANN/MindSpore 软件栈、超节点、网络互联、液冷电源、算力运营、行业应用。

第二张是“A股交易链”：卖方和资金把昇腾主题扩散到服务器、交换机、NPO/光互连、连接器、液冷、电源、PCB、算力租赁、软件适配等环节。

这两张图重叠的地方，才是最值得研究的地方。

一句话结论：

> 华为昇腾链的核心价值在华为，A股主要交易的是“超节点外溢”和“国产算力叙事扩散”。最值得优先研究的，不是泛华为概念，而是卖方叙事、产业瓶颈、A股可表达三者重叠的环节：NPO/光互连、高速连接器、服务器/交换机、液冷/电源、软件适配和算力运营。

需要特别注意：很多公司只是“AI 算力基础设施受益”，不一定真在“华为昇腾产业链”里。因此本报告对公司使用关系证据等级：

| 等级 | 含义 | 使用方式 |
|---|---|---|
| L3 | 官方/公告/中标/年报确认 | 可列为确认产业链公司 |
| L2 | 公司公开互动、生态认证、合作活动等支持 | 可列为较高可信映射 |
| L1+ | 多个券商/zsxq 反复提及 | 高优先级卖方候选 |
| L1 | 单一券商/社群/媒体提及 | 叙事候选 |
| L0 | 只是产品逻辑相关 | 只能列为产业外溢，不能叫昇腾确认链 |

## 2. 产业边界：华为昇腾链包括什么？

华为昇腾链不是普通硬件供应链，而是一个“国产 AI 算力全栈生态”。

昇腾生态包括：

- Atlas 硬件；
- CANN 异构计算架构；
- MindSpore；
- MindSpeed；
- MindIE；
- MindCluster；
- MindStudio；
- MindSDK；
- 面向互联网、金融、城市、教育、能源、交通、制造、医疗等行业应用的解决方案。

产业地图可以这样画：

```text
终端AI需求
  -> 大模型训练 / 推理 / 行业AI
  -> 算力集群 / 超节点 / 云算力
  -> Atlas服务器 / SuperPoD / 交换网络
  -> 昇腾NPU + CANN + MindSpore + MindIE + MindSpeed
  -> 光互连 / 交换机 / PCB / 连接器 / 液冷 / 电源 / 机柜
  -> 上游芯片制造 / 封装 / 存储 / 材料 / 设备
```

核心区别：

- 昇腾链核心：华为控制，A股直接映射少；
- 昇腾链生态：服务器、软件适配、行业落地，有 A股映射；
- AI算力外溢：液冷、PCB、连接器、光模块、电源等，不一定是昇腾确认供应链，但会被市场一起交易。

## 4. 六问分析

### 4.1 体量、增长、周期位置

昇腾链的增长来源不是单一的 AI 芯片替代，而是三股力量叠加。

第一，外部限制倒逼国产算力。美国对先进 AI 芯片出口限制，使国内高端训练和推理需要国产替代方案。华为昇腾因此拥有战略地位。

第二，大模型从训练走向推理。训练侧更看单卡性能和集群效率，推理侧更看成本、稳定性、部署、生态适配。国产算力在政企、运营商、金融、能源等本地部署场景更容易落地。

第三，超节点成为新叙事。卖方材料近期集中强调 Atlas 950、SuperPoD、8192卡、服务器+交换机、NPO/光互连。这说明市场从“买 AI 服务器”进一步扩散到“买超节点基础设施”。

当前阶段定义为：

```text
政策驱动和供给突破阶段
  -> 进入超节点事件催化阶段
  -> 后续必须进入订单和财务验证阶段
```

换句话说：叙事已经很热，下一步要看业绩兑现。

### 4.2 价值链结构与利润分布

昇腾链利润池不是平均分配。

| 利润池 | 位置 | 谁可能拿走利润 | A股能否直接表达 |
|---|---|---|---|
| 第一利润池 | 昇腾芯片、CANN、Atlas、华为云 | 华为 | 很弱 |
| 第二利润池 | 超节点、交换机、光互连标准、整机方案 | 华为生态、ICT设备商 | 部分可表达 |
| 第三利润池 | NPO/光模块/连接器/液冷/电源/PCB | 供应链厂商 | 较强 |
| 第四利润池 | 算力租赁、IDC、行业应用 | 运营商、IDC、软件集成商 | 强但质量分化 |
| 第五利润池 | 低毛利代理/分销/普通系统集成 | 渠道商 | 收入弹性有，利润质量弱 |

关键结论：

> A股买到的通常不是第一利润池，而是第三、第四、第五利润池。因此不能只看“昇腾空间大”，要看上市公司到底在哪个利润池。

### 4.3 护城河与竞争优势

华为的护城河很强，但 A股伙伴的护城河不一定强。

华为的护城河：

- 芯片、服务器、云、软件栈、开发工具一体化；
- 政企、运营商、金融等客户基础；
- 国产替代战略地位；
- 工程化组织能力；
- 全栈生态绑定。

A股公司的护城河需要单独判断：

| 公司类型 | 真护城河 | 伪护城河 |
|---|---|---|
| 服务器/OEM | 大客户认证、交付能力、规模制造、交换网络能力 | 只是代理华为产品 |
| 软件适配 | 可复用产品、行业模板、开发工具能力 | 人力外包/项目制 |
| 光互连/NPO | 标准参与、量产能力、客户认证、良率 | 只蹭CPO/NPO概念 |
| 连接器 | 高速产品、核心客户、认证周期 | 普通连接器业务 |
| 液冷 | 大客户订单、方案能力、产品化 | 一次性项目 |
| 算力运营 | 高利用率、长约客户、低电价、低融资成本 | 买设备堆资产 |

### 4.4 技术演化与颠覆风险

昇腾技术演化的核心不是单芯片，而是“系统级补偿单点性能差距”。

华为路线强调 Atlas 950/960 SuperPoD 这类超节点/超集群，用更多 NPU、更强互联和系统工程去弥补单芯片与海外最先进 GPU 的差距。公开报道也提到，华为试图通过更大规模互联和国内芯片优化来应对美国限制下的先进芯片短板。

技术变量包括：

| 技术变量 | 影响 |
|---|---|
| 单芯片性能 | 决定训练效率和推理成本下限 |
| 集群互联 | 决定超节点效率，带动交换机、光模块、NPO、连接器 |
| CANN生态 | 决定模型迁移和开发者体验 |
| MindSpore/PyTorch适配 | 决定开发者是否愿意迁移 |
| 推理优化 | 决定国产算力商业化落地速度 |
| 液冷/电源 | 决定高密度部署能否规模化 |
| HBM/存储/先进封装 | 决定供给瓶颈 |

最大风险是：如果软件生态长期难用，昇腾可能只能在政策和安全可控场景替代，而难以成为开发者主动选择的通用生态。

### 4.5 监管、地缘、国产替代

这是昇腾链最核心的外生变量。

美国先进 AI 芯片出口限制强化了国产算力需求。这个变量使昇腾链具备强政策底色，也让市场愿意给它较高叙事权重。

但国产替代不等于利润平均分配：

- 华为拿走核心控制权；
- 整机厂可能拿收入，但毛利率未必高；
- 基础设施供应商能否赚钱，看产品壁垒和客户认证；
- 算力运营商能否赚钱，看利用率和租赁价格；
- 软件适配商能否赚钱，看是否产品化，而不是人力项目制。

### 4.6 估值、叙事与数字

当前市场叙事已经很清晰：

```text
Atlas 950 / 超节点事件催化
  -> 国产算力2026H2新方向
  -> 服务器 + 交换机
  -> OPEN NPO / 光互连
  -> 连接器、液冷、电源、PCB
  -> 算力租赁和软件适配
```

这套叙事要成立，数字上必须看到：

- 订单，而不是框架合作；
- 收入占比，而不是“参与生态”；
- 毛利率，而不是低毛利代理；
- 现金流，而不是应收账款；
- 产能利用率，而不是扩产故事；
- 估值没有过度透支。

## 5. zsxq/卖方叙事雷达

本地 zsxq 近期最有价值的信息，不是证明谁拿了订单，而是显示市场正在如何组织昇腾链。

### 5.1 线索一：Atlas 950 / SuperPoD / 超节点

卖方材料集中强调 2026H2 国产算力新方向，关注 Atlas 950、超节点、服务器和交换机机遇。

叙事含义：

```text
昇腾不再只是AI芯片
  -> 变成超节点系统工程
  -> 服务器、交换机、互联、液冷、电源、连接器一起被纳入
```

### 5.2 线索二：OPEN NPO / 光互连

zsxq 材料反复提到华为联合产业伙伴启动 OPEN NPO 项目，市场关注光互连标准、NPO、光进铜退。

被点名较多的方向包括：

- 光迅科技；
- 华工科技/华工正源；
- 华丰科技；
- 立讯精密；
- 新华三/紫光股份；
- 中国移动研究院等生态角色。

这类材料如果得到官方/产业新闻交叉验证，可以从 L1+ 叙事候选升到 L2 生态候选。

### 5.3 线索三：连接器/高速模组

材料中出现永贵电器、航天电器、华丰科技等方向，逻辑集中在高速连接、液冷冷板、BUSBAR、高速模组等。

这条线的关键是：连接器是超节点和光互连的基础件，但公司是否进入华为/超聚变/Atlas 体系，需要公告、客户验证或招股书信息进一步确认。

### 5.4 线索四：算力运营和软件映射

慧博云通被部分材料映射为 Atlas 950 软件/服务方向；数据港、网宿科技、深信服等被纳入算力租赁、华为算力、信息安全等交易链。

这类是叙事弹性强、财务验证难度大的方向。适合放入候选池，但不能直接当确认供应链。

## 7. 优先研究环节

### 7.1 第一优先级：NPO/光互连 + 高速连接器

原因：这是 zsxq 近期最集中的卖方叙事，也是超节点从服务器扩散到互联的核心方向。

优先候选：

- 光迅科技；
- 华工科技；
- 华丰科技；
- 永贵电器；
- 航天电器；
- 立讯精密；
- 紫光股份/新华三相关链条。

为什么排第一？

```text
产业瓶颈：超节点需要更高效互联
卖方叙事：近期OPEN NPO材料密集
事件催化：Atlas 950 / WAIC / 超节点
A股表达：有可交易标的
```

风险：很多说法仍是卖方/社群材料，需要官方名单、订单、认证进一步确认。

### 7.2 第二优先级：服务器 + 交换机 + 超节点OEM

风险：服务器链容易收入大、毛利薄；还要区分“国产AI服务器”与“华为昇腾链”的真实关系。

## 8. 公司筛选方法：四步走

以后昇腾链不应直接列股，而应按四步筛。

### 第一步：是否被卖方/市场纳入候选？

如果 zsxq 多次提及，先进入 L1+ 候选池。

### 第二步：是否有公开关系证据？

查：

- 华为官方生态；
- 昇腾社区；
- 华为开发者大会；
- OPEN NPO 官方发起单位；
- 公司公告/年报；
- 投资者关系；
- 互动易；
- 招投标/中标；
- 伙伴认证。

有证据则升级 L2/L3。

### 第三步：业务纯度和利润质量

问：

- 相关收入占比多少？
- 是代理/分销，还是自有产品？
- 毛利率是否高于公司平均？
- 是否有持续订单？
- 是否形成复购？

### 第四步：估值和拥挤度

问：

- 股价是否已经按核心昇腾链定价？
- 盈利预测是否上修？
- 成交是否过热？
- 是否只有叙事没有财报？

## 12. 最终判断

这次重做后，对华为昇腾链的判断更清楚了：

第一，昇腾链是强产业主线。它有国产替代、AI算力缺口、华为全栈生态、政企需求和外部限制五个驱动力。

第二，A股不能直接买华为核心利润池。所以研究重点要从“芯片”转向“超节点外溢”：服务器、交换机、NPO、连接器、液冷、电源、PCB、算力运营、软件适配。

第三，zsxq/券商材料应该显著提高候选名单权重。本地材料显示市场最热的扩散方向是 Atlas 950、超节点、OPEN NPO、光互连、连接器、服务器+交换机。这里要重点研究。

第四，最优先环节不是泛液冷/泛PCB，而是“卖方叙事和产业瓶颈重叠”的部分：

```text
NPO/光互连
  + 高速连接器
  + 服务器/交换机
  + 超节点液冷/电源
```

第五，所有公司必须标关系证据等级。目前很多公司只能是 L1+ 或 L1，不能直接写成“确认昇腾链”。

优先研究顺序：

1. 光迅科技、华工科技、华丰科技、永贵电器、航天电器这一组 NPO/光互连/连接器候选；
2. 紫光股份、锐捷网络、浪潮信息、中科曙光、中兴通讯这一组服务器/交换机/超节点候选；
3. 申菱环境、英维克、高澜股份、科华数据、科士达、金禄电子、沪电股份、胜宏科技这一组 AI基础设施外溢；
4. 慧博云通、软通动力、拓维信息、常山北明、数据港、网宿科技、深信服这一组软件/算力运营高弹性候选。

如果只选“最值得先查”的公司：

```text
光迅科技、华工科技、华丰科技、永贵电器、航天电器、紫光股份、锐捷网络、浪潮信息、申菱环境、慧博云通
```

不是说它们一定最好，而是它们最集中地处在“卖方叙事 + 产业瓶颈 + A股可表达”的交叉点。
