# 国产算力产业地图分析报告

> 生成日期：2026-07-13  
> 分析框架：A股产业地图六问 + 卖方叙事雷达 + 关系证据等级  
> 研究对象：国产 AI 算力、国产芯片、服务器/超节点、互联、液冷电源、算力运营、软件生态  
> 结论用途：研究支持，不构成证券投资建议

---

## 0. 核心结论

国产算力是比“华为昇腾链”更大的产业。昇腾只是国产算力的一条核心主线，除此之外还包括海光、寒武纪、沐曦、壁仞、天数智芯、燧原、摩尔线程等 AI/GPU/DCU 芯片路线，以及服务器、交换机、互联、液冷、电源、PCB、光模块、先进封装、算力运营和模型应用生态。

一句话结论：

> 国产算力的核心矛盾，是中国大模型和政企 AI 需求快速增长，但高端 Nvidia 供给受限，因此国内必须用“国产芯片 + 集群工程 + 软件生态 + 基础设施卖铲子”来补齐算力缺口。A股最值得研究的不是泛 AI 概念，而是三个交叉点：国产 AI 芯片、超节点/互联基础设施、以及能够把算力放量转成真实收入和利润的服务器/运营/应用公司。

本报告延续关系证据等级：

| 等级 | 含义 | 使用方式 |
|---|---|---|
| L3 | 官方/公告/中标/年报确认 | 可列为确认产业链公司 |
| L2 | 公司公开互动、生态认证、合作活动等支持 | 可列为较高可信映射 |
| L1+ | 多个券商/zsxq 反复提及 | 高优先级卖方候选 |
| L1 | 单一券商/社群/媒体提及 | 叙事候选 |
| L0 | 只是产品逻辑相关 | 只能列为产业外溢，不能叫确认链 |

## 1. 产业边界

国产算力不是单一芯片行业，而是一整套 AI 基础设施体系。

| 层级 | 内容 | 代表方向 |
|---|---|---|
| 算力芯片 | GPU、NPU、DCU、AI ASIC、CPU、互联芯片 | 昇腾、寒武纪、海光、沐曦、壁仞、燧原、摩尔线程、盛科、澜起等 |
| 芯片制造/封装 | 晶圆制造、先进封装、Chiplet、HBM/存储、测试 | 中芯、华虹、长电、通富、甬矽、长鑫链等 |
| 硬件整机 | AI服务器、训练服务器、推理服务器、超节点、交换机 | 浪潮信息、紫光股份、锐捷网络、中科曙光、华勤技术等 |
| 集群互联 | 高速交换、光模块、NPO/CPO、连接器、PCB | 光迅、华工、华丰、沪电、胜宏、中际旭创、新易盛等 |
| 数据中心基础设施 | 液冷、温控、电源、机柜、IDC | 英维克、申菱环境、高澜、科华、科士达、润泽、数据港等 |
| 软件生态 | 驱动、编译器、算子库、框架适配、CUDA迁移 | CANN、DTK、各芯片厂软件栈、模型适配商 |
| 算力运营 | 智算中心、云算力、算力租赁、政企算力平台 | 运营商、地方算力平台、IDC、云厂商 |
| 应用需求 | 大模型训练、推理、AI Coding、Agent、政企行业AI | 互联网、政务、金融、能源、教育、制造、医疗 |

这条链有一个重要特点：上游芯片价值最高，但不一定最容易赚钱；中游硬件放量最快，但毛利率可能薄；下游算力运营空间大，但重资产风险高。

## 2. 产业地图

```text
AI应用需求
  -> 大模型训练 / 推理 / Agent / AI Coding / 行业AI
  -> 云厂商、运营商、政企、模型公司采购算力
  -> 国产AI芯片 / 国产GPU / DCU / NPU
  -> AI服务器 / 超节点 / 交换机 / 集群互联
  -> 光模块 / NPO / 连接器 / PCB / 液冷 / 电源
  -> 晶圆制造 / 先进封装 / 存储 / 材料 / 设备
```

| 节点 | 功能 | 利润池质量 | 主要玩家/映射 | 判断 |
|---|---|---|---|---|
| 国产AI芯片 | 算力核心 | 最高，但研发和制造风险大 | 华为昇腾、寒武纪、海光、沐曦、壁仞、摩尔线程、天数智芯等 | 产业战略价值最高 |
| CPU/DCU/互联芯片 | 训练/推理调度、通用计算、集群互联 | 高 | 海光信息、盛科通信、澜起科技、裕太微等 | 国产算力从单卡走向系统级竞争 |
| 先进制造/封装 | 决定供给和良率 | 高，但约束多 | 中芯国际、华虹、长电科技、通富微电、甬矽电子等 | 国产算力真正瓶颈之一 |
| AI服务器/超节点 | 把芯片变成集群算力 | 中高 | 浪潮信息、紫光股份、中科曙光、锐捷网络、华勤技术等 | 收入弹性强，毛利率需验证 |
| 交换机/网络 | 集群通信 | 高 | 紫光股份、锐捷网络、盛科通信等 | 超节点时代重要性上升 |
| 光互连/NPO/CPO | 降功耗、提带宽 | 高 | 光迅科技、华工科技、华丰科技、中际旭创、新易盛、天孚通信等 | 卖方叙事很热，需区分国产链和海外链 |
| PCB/连接器 | 高速信号传输 | 中高 | 沪电股份、胜宏科技、深南电路、华丰科技、永贵电器、航天电器 | AI服务器硬件升级核心配套 |
| 液冷/电源 | 高功耗算力中心基础设施 | 中高 | 英维克、申菱环境、高澜股份、科华数据、科士达 | 产业逻辑硬，但不一定是某芯片链确认供应商 |
| 算力运营/IDC | 采购设备并出租算力 | 分化大 | 运营商、润泽科技、数据港、奥飞数据、网宿科技等 | 空间大，但重资产风险最大 |
| 软件适配/生态 | 降低迁移成本 | 分化大 | 软通动力、拓维信息、慧博云通等 | 真瓶颈，但商业模式可能项目制 |

## 3. 六问分析

### 3.1 体量、增长、周期位置

国产算力的需求来自四股力量。

第一，外部限制。美国持续限制中国获得先进 AI 芯片和相关半导体能力。美国 BIS 2023 年规则明确强化对中国先进计算芯片和先进芯片制造能力的限制。

第二，国内 AI 模型增长。大模型、AI Coding、Agent、推理场景带来持续算力需求。

第三，政策推动。我国《算力基础设施高质量发展行动计划》提出到 2025 年算力规模超过 300 EFLOPS，智能算力占比达到 35%。

第四，政企国产化。金融、运营商、政务、能源等领域有安全可控、本地部署、国产化采购需求。

当前周期不是简单的“概念期”，而是从概念期进入订单验证期。

市场现在交易的是：

```text
国产芯片可用
  -> 国产大模型适配
  -> 智算中心建设
  -> 超节点/互联升级
  -> 服务器、交换机、光互连、液冷、电源、PCB放量
```

但财务上仍要验证：订单、毛利率、现金流、产能利用率。

### 3.2 价值链和利润分布

国产算力的利润池可以分五层：

| 利润池 | 环节 | 质量 | 风险 |
|---|---|---|---|
| 第一利润池 | AI芯片、软件栈、生态控制点 | 最高 | 研发投入大、供给受限、生态难 |
| 第二利润池 | 超节点、交换机、互联芯片、系统方案 | 高 | 客户认证和集群稳定性难 |
| 第三利润池 | 光互连、连接器、PCB、液冷、电源 | 中高 | 容易被主题炒热，需看订单 |
| 第四利润池 | AI服务器/OEM/集成 | 收入弹性强 | 毛利率可能薄 |
| 第五利润池 | 算力租赁/IDC/行业项目 | 空间大但分化 | 重资产、利用率、回款、价格战 |

所以国产算力里最不能犯的错误是：看到“算力需求大”，就认为所有环节都赚钱。真正赚钱的是瓶颈，不是所有参与者。

### 3.3 护城河和竞争优势

国产算力公司的护城河分四类：

| 护城河 | 代表环节 | 如何验证 |
|---|---|---|
| 架构和软件生态 | AI芯片、DCU、NPU、CUDA迁移 | 主流模型适配、开发者生态、客户复购 |
| 工程化交付 | 服务器、超节点、交换机 | 集群稳定性、交付规模、客户案例 |
| 供应链卡位 | 先进封装、PCB、连接器、液冷 | 客户认证、良率、毛利率、产能利用率 |
| 运营能力 | 算力租赁、IDC | 利用率、租赁价格、客户合同、电价 |

伪护城河包括：

- 只是“国产算力概念”；
- 只做低毛利代理；
- 只有框架协议没有订单；
- 收入增长但毛利率和现金流下降；
- 只是卖方材料反复提及，但无公司披露验证。

### 3.4 技术演化和颠覆风险

国产算力的技术路线正在从“单卡性能追赶”转向“系统级能力追赶”。

| 技术方向 | 产业影响 |
|---|---|
| 单卡性能提升 | 决定训练和推理效率基础 |
| 超节点/集群互联 | 用系统工程弥补单卡差距 |
| FP4/FP8等低精度 | 提升推理效率，降低成本 |
| NPO/CPO/光互连 | 降低互联功耗，提高集群带宽 |
| 液冷和高密度部署 | 支撑更大规模智算中心 |
| 软件栈适配 | 决定国产芯片能否被开发者真正使用 |
| 先进封装/Chiplet | 在先进制程受限下提升系统性能 |

最大的技术风险是：硬件能交付，但软件生态不好用。  
如果 CUDA 迁移成本太高，国产算力就只能在政策刚性场景使用，难以在互联网和模型公司中形成主动替代。

### 3.5 监管、地缘和国产替代

国产算力是典型的政策和地缘驱动行业。

外部限制提升国产替代必要性；国内政策提升算力基础设施建设确定性。但国产替代不自动等于利润。

替代阶段可分为：

| 环节 | 替代阶段 | 判断 |
|---|---|---|
| AI芯片 | 从验证走向规模使用 | 华为、海光、寒武纪等路线开始分化 |
| 服务器/超节点 | 规模化建设初期 | 最容易先体现收入 |
| 软件生态 | 从能用到好用 | 是最大短板之一 |
| 光互连/连接器/液冷 | 随超节点建设加速 | 当前卖方叙事热度高 |
| 算力运营 | 快速扩张但质量分化 | 重资产风险大 |
| 先进制造/封装 | 战略瓶颈 | 供给和良率决定上限 |

### 3.6 估值、叙事和数字

当前市场叙事是：

```text
海外AI芯片受限
  -> 国产算力必须加速
  -> 国产模型能力提升
  -> 国产芯片进入订单兑现期
  -> 超节点和互联成为新方向
  -> A股扩散到芯片、服务器、交换机、光互连、液冷、电源、PCB、算力租赁
```

这套叙事要成立，需要数字验证：

- 芯片公司收入持续增长；
- 服务器/交换机公司订单增长且毛利率不被压；
- 光互连/连接器/PCB/液冷公司相关业务占比提升；
- 算力运营利用率高；
- 软件适配不是纯人力项目；
- 股价上涨伴随盈利预测上修，而不是只涨估值。

## 4. zsxq 卖方叙事雷达

本地 zsxq 里，近期国产算力叙事非常集中，主要有四条线。

### 4.1 线索一：国产算力芯片

反复出现：

- 海光信息；
- 寒武纪；
- 沐曦；
- 天数智芯；
- 中兴 AI 芯片；
- 国产 GPU / DCU；
- 国产模型和国产芯片正循环。

有材料提到“国产算力 3+3”：

```text
国产算力芯片：寒武纪、海光信息、盛科通信
国产互联芯片：澜起科技、万通发展、裕太微
```

这是卖方叙事候选，不是事实确认，但说明市场正在把国产算力从“单芯片”扩展到“互联芯片”。

### 4.2 线索二：超节点 OEM / 服务器 / 交换机

zsxq 中明确出现：

- 浪潮信息；
- 紫光股份；
- 中科曙光；
- 锐捷网络；
- 华勤技术；
- 中兴通讯；
- 联想集团。

叙事逻辑是：AI 推理 + 国产算力双轮驱动，超节点 OEM 迎来收入、利润率、估值重估。

### 4.3 线索三：NPO / 光互连 / 光进铜退

近期热度很高，和华为 OPEN NPO、Atlas 950、超节点叙事结合。

被提到较多：

- 光迅科技；
- 华工科技；
- 华丰科技；
- 立讯精密；
- 天孚通信；
- 中际旭创；
- 新易盛；
- 源杰科技；
- 长光华芯；
- 仕佳光子。

其中要区分：

- 光迅、华工、华丰：更容易被市场映射到国产算力/华为链；
- 中际旭创、新易盛、天孚：更偏全球 AI 光模块链，不等于国产算力确认供应链。

### 4.4 线索四：先进封装、液冷、电源、PCB

被反复带出的方向包括：

- 先进封测：长电科技、通富微电、甬矽电子；
- 液冷/散热：申菱环境、英维克、高澜股份、中石科技；
- PCB：沪电股份、胜宏科技、金禄电子、深南电路；
- 电源/电容：江海股份、科华数据、科士达等。

这类是国产算力“外溢基础设施”，产业逻辑强，但需要验证是否进入具体国产算力链。

## 6. 优先研究环节

如果按“产业逻辑 + 卖方叙事 + A股可表达”三者交集，优先级如下。

### 6.1 第一优先级：国产 AI 芯片与互联芯片

核心候选：

- 寒武纪；
- 海光信息；
- 盛科通信；
- 澜起科技；
- 裕太微。

原因：这是国产算力最核心的利润池。芯片和互联芯片决定国产算力能不能从单点可用走向集群可用。

风险：估值高、研发投入大、供给受限、生态验证周期长。

### 6.2 第二优先级：超节点 OEM / 服务器 / 交换机

核心候选：

- 浪潮信息；
- 紫光股份；
- 中科曙光；
- 锐捷网络；
- 中兴通讯；
- 华勤技术。

原因：这是国产算力从芯片走向收入的第一落点。服务器和交换机最容易先看到订单。

风险：收入大但毛利薄；要防止低毛利集成。

### 6.3 第三优先级：NPO/光互连/连接器

核心候选：

- 光迅科技；
- 华工科技；
- 华丰科技；
- 永贵电器；
- 航天电器；
- 立讯精密。

原因：超节点时代，互联瓶颈上升。近期卖方叙事热度也最高。

风险：很多公司仍处于 L1/L1+，需要官方/订单证据升级。

### 6.4 第四优先级：先进封装 + PCB + 液冷电源

核心候选：

- 长电科技、通富微电、甬矽电子；
- 沪电股份、胜宏科技、深南电路、金禄电子；
- 英维克、申菱环境、高澜股份；
- 科华数据、科士达。

原因：这些是算力建设绕不开的卖铲子。

风险：很多不是国产算力专属，而是 AI算力外溢；估值也可能已被全球 AI 链透支。

### 6.5 第五优先级：算力运营和软件适配

核心候选：

- 润泽科技、数据港、奥飞数据、网宿科技；
- 软通动力、拓维信息、慧博云通、常山北明。

原因：如果国产算力规模化，运营和适配会有需求。

风险：最容易出现“收入有、利润少、现金流差”的问题。

## 7. 最需要警惕的误区

### 7.1 误区一：把 AI算力外溢当成国产算力确认供应链

例如液冷、PCB、光模块、电源公司，确实受益 AI 算力建设，但不一定直接进入国产芯片/国产算力链。必须标 L0/L1/L2/L3。

### 7.2 误区二：只看收入，不看毛利率

服务器和算力集成容易出现收入大增、利润不增。国产算力主题最怕低毛利堆收入。

### 7.3 误区三：把算力运营当成轻资产

算力租赁听起来很好，但实际要看：

- 设备采购成本；
- 折旧周期；
- 电价；
- 利用率；
- 客户长约；
- 租赁价格是否下行。

### 7.4 误区四：忽略软件生态

国产芯片能不能用，不只看硬件参数，还要看：

- PyTorch/CUDA 迁移；
- 算子库；
- 编译器；
- 主流模型适配；
- 推理性能；
- 开发者体验。

### 7.5 误区五：叙事太热后追高

国产算力是强主线，但越强的主线越容易阶段性拥挤。需要用财报验证叙事。

## 9. 证伪条件

国产算力主线会被这些信号证伪或阶段性削弱：

- 国产芯片出货不及预期；
- 主流模型适配困难；
- 服务器公司收入增长但毛利率下滑；
- 算力租赁利用率低；
- 超节点/NPO 只有概念没有订单；
- 先进封装/存储/制造瓶颈迟迟无法缓解；
- 股价大涨但盈利预测不上修；
- 政策采购低价化，导致国产替代只带来收入不带来利润。

## 10. 最终判断

国产算力是 A股 AI 主线里最重要的一条，不只是因为“国产替代”，而是因为它同时具备：

- 外部限制倒逼；
- 国内大模型需求；
- 政企安全可控；
- 算力基础设施政策支持；
- 超节点和互联技术演进；
- A股有足够多可表达标的。

但产业地图告诉我们：这条链不能简单买“国产算力概念”。要按利润池和证据等级分层。

优先研究顺序：

1. 国产 AI 芯片和互联芯片：寒武纪、海光信息、盛科通信、澜起科技、裕太微；
2. 服务器/交换机/超节点 OEM：浪潮信息、紫光股份、中科曙光、锐捷网络、中兴通讯；
3. NPO/光互连/连接器：光迅科技、华工科技、华丰科技、永贵电器、航天电器；
4. 先进封装/PCB/液冷电源：长电科技、通富微电、甬矽电子、沪电股份、胜宏科技、英维克、申菱环境、科华数据；
5. 算力运营/软件适配：润泽科技、数据港、网宿科技、软通动力、拓维信息、慧博云通。

其中，最值得先深入验证的一组是：

```text
寒武纪、海光信息、浪潮信息、紫光股份、光迅科技、华工科技、华丰科技、沪电股份、英维克、润泽科技
```

不是说它们一定最好，而是它们分别代表国产算力的核心芯片、硬件放量、互联瓶颈、基础设施和运营变现五个关键环节。
