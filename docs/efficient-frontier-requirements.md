# 有效前沿（Efficient Frontier）功能需求

## 一、核心概念

根据马科维茨均值-方差理论，基于你的持仓股票的历史数据：
1. 估计每只股票的**年化预期收益率**
2. 估计收益率之间的**协方差矩阵**  
3. 对不同目标收益率，解优化：**最小化组合方差，满足收益率 = 目标值**
4. 把所有最优解画在 (风险, 收益) 坐标上 → 就是**有效前沿曲线**

## 二、数据源

从 Tushare 拉最近 **250 个交易日**的日收益率 (`pct_chg`)，计算：
- 年化预期收益率 = 日均收益率 × 252
- 年化协方差矩阵 = 日协方差 × 252

## 三、优化问题（对每个目标收益率 R_target）

```
最小化:  w^T Σ w          （组合方差）
约束:
  ①  Σ w_i = 1            （权重合计 100%）
  ②  w_i ≥ 0              （不允许做空）
  ③  w_i ≤ 0.4            （单只股票上限 40%，防止过度集中）
  ④  w^T μ ≥ R_target     （预期收益不低于目标）
```

用**二次规划（Quadratic Programming）**求解。

## 四、前端展示

在现有页面上新增一个**「有效前沿」标签页或卡片**，包含：

### 4.1 有效前沿散点图（ECharts）
- X 轴：年化波动率（标准差 %）
- Y 轴：年化预期收益率（%）
- 蓝色曲线：有效前沿（20 个目标收益水平的最优解连线）
- 红色散点：9 只单只股票各自的位置（收益 vs 波动）
- 黄色散点：**你当前持仓的实际位置**（按实际持有市值的权重）

### 4.2 最优组合表
- 用户可以选择目标收益率（滑条或点击前沿上的点）
- 表格显示该目标收益下每只股票的建议权重
- 与当前持仓权重对比（增量/减量）

### 4.3 关键指标卡片
- 最小方差组合（MVP）的收益/风险
- 最大 Sharpe 比率组合的收益/风险（假设无风险利率 2.5%）
- 当前持仓的收益/风险/Sharpe

## 五、后端 API

**GET /api/portfolio/efficient-frontier**
- 参数：`ts_codes`, `days`（默认 250），`risk_free`（默认 2.5%）
- 返回：有效前沿数据点 + 最小方差组合 + 最大 Sharpe 组合 + 当前持仓位置
