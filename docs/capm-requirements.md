# CAPM 分析功能需求

## 一、核心概念

根据 CAPM（资本资产定价模型）：

```
个股日收益率 r_i = α_i + β_i × r_m + ε_i
```

- **β (Beta)**：个股相对大盘的系统性风险敏感度
  - `β > 1`：波动比大盘大（进攻型）
  - `β < 1`：波动比大盘小（防御型）
  - `β < 0`：与大盘反向
- **α (Alpha)**：剔除市场波动后，个股自身的超额收益
  - `α > 0`：跑赢大盘同类风险资产
  - `α < 0`：跑输
- **非系统性风险**：ε 的标准差，即无法被大盘解释的"个股自身波动"

```
组合层面（按持仓权重加权）：
  组合 β_p = Σ(w_i × β_i)
  组合 α_p = Σ(w_i × α_i)
  组合非系统风险 = sqrt(Σ(w_i² × σ_εi²))
```

## 二、数据源

### 2.1 市场代理（待确认）
对每只股票 + 市场指数，拉最近 **250 日**日线数据，取 `pct_chg`。
做 OLS 回归：`个股收益率 ~ 市场收益率`。

### 2.2 计算步骤
```
① 对齐交易日 → 个股日收益率序列 + 市场日收益率序列
② 线性回归：r_i = intercept + slope × r_m → α_daily = intercept, β = slope
③ 年化：α_annual = α_daily × 252
④ 非系统风险：σ_ε = std(residuals) × √252
⑤ 统计检验：t 值 / p 值 判断 α 是否统计显著
⑥ R²：市场能解释个股多少波动
```

结论自动生成：
- α显著正 + β适中 → "优质超额收益"
- α显著负 → "⚠️ 跑输大盘"
- β极高 → "🚀 高Beta进攻型"
- R²低 → "与大盘关联弱"

- 组合系统性风险 = β_p² × σ_m²
- 组合总风险 = 系统性 + 非系统性
